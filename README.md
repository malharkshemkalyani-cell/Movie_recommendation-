# Movie_recommendation-
This project demonstrates how recommendation systems works . taking movies as an example to  recommend . this project explores two main approaches of the recommendation system ie. content based recommendation system and user based recommendation system 
### Recommendation Systems: User-Based vs. Content-Based

| Feature | User-Based Collaborative Filtering | Content-Based Filtering |
| **Core Logic** | "Users who liked this also liked..." | "Because you liked item A, you might like item B..." |
| **Data Source** | Relies on user behavior, ratings, and interactions. | Relies on item attributes (e.g., engine size, fuel type). |
| **Similarity** | Finds similar **users** based on their shared preferences. | Finds similar **items** based on their specific features. |
| **Cold Start** | Struggles with new users who have no history. | Struggles with new items until features are defined. |
| **Diversity** | Can recommend niche items outside a user's usual scope. | Recommendations are limited to what the user has seen before. |
| **Example** | Recommending a toy story to a user because similar users bought toy story | Recommending action film because the user previously viewed action film |
